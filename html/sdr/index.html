You will find here a collection of tools you can use for radio tracking, using a software defined radio. For further details please either contact me or have a look at the documentation radio-tracking.eu.

This Project is developed using the latest MIT Licence and comes with absolutly no warrenty.

Raspberry-Pi SDR Tools Version 1.1:

1: Simple Receiver Advanced Receiver Like a radio receiver, just set the frequency you want to listen to.
2: RTL_TCP Server Sets up a server which can be used to monitor a whole frequency range.
3: Signal Recorder Advanced Recorder Records the signals in a specified frequency range.
4: Enviromental Data Records: GPS Position, Temperature,... just works if sensors are connected. Please connect first a USB GPS Device, tested using Navilok NL-602U.
5: Record Folder Folder where the recorded data is stored.
6: Cross Bearing: external Link Application which calculates the corss bearing intersections
